# birdwatch-public-data

This is an archive of X(Twitter)'s _birdwatch-public-data_ text files.

# URL Format

```
https://media.githubusercontent.com/media/xdatacurate/birdwatch-public-data/refs/heads/main/data/{YYYY}/{MM}/{DD}/data.{EXT}
```
- Replace `{}`;
  - `{YYYY}` year, `{MM}` month, `{DD}` day
  - `{EXT}` is either `txt` (for text format) or `7z` (compressed archive)
- Look at `latest.txt` to know latest uploaded date.
